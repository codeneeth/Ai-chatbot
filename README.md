## 🔧 Setup Instructions

```bash
# 1. Clone the repo
git clone https://github.com/codeneeth/ai-chatbot.git
cd ai-chatbot

# 2. Install dependencies
bun install

# 3. Set up environment
echo 'VITE_GEMINI_API=your-key-here' > .env

# 4. Run locally
bun run dev
